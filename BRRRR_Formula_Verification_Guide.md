## üìä **1. BASIC PROPERTY CALCULATIONS**

### **Price per Square Foot**
```
Formula: = Listing Price √∑ Square Footage
Example: $699,000 √∑ 2,500 sqft = $279.60/sqft
Verification: $279.60 √ó 2,500 = $699,000 ‚úì
```

### **Lot Size in Acres**
```
Formula: = Lot Size (sqft) √∑ 43,560
Example: 9,125 sqft √∑ 43,560 = 0.21 acres
Verification: 0.21 √ó 43,560 = 9,125 sqft ‚úì
```

## üè† **2. ARV (AFTER REPAIR VALUE) CALCULATIONS**

### **Base ARV Calculation**
```
Formula: = Square Footage √ó ARV per sqft √ó ARV Multiplier
Example: 2,500 √ó $450 √ó 1.0 = $1,125,000
Verification: $1,125,000 √∑ 2,500 = $450/sqft ‚úì
```

### **ARV Increase (Forced Equity)**
```
Formula: = ARV - Listing Price
Example: $1,125,000 - $699,000 = $426,000
Verification: $699,000 + $426,000 = $1,125,000 ‚úì
```

### **ARV per Square Foot (Final)**
```
Formula: = ARV √∑ Square Footage
Example: $1,125,000 √∑ 2,500 = $450/sqft
Verification: $450 √ó 2,500 = $1,125,000 ‚úì
```


### **Repair Cost Percentage of ARV**

## üí∞ **4. FINANCING CALCULATIONS**

### **Down Payment Amount**
```
Formula: = Listing Price √ó Down Payment Percentage
Example: $699,000 √ó 20% = $139,800
Verification: $139,800 √∑ $699,000 = 20% ‚úì
```

### **Loan Amount**
```
Formula: = Listing Price - Down Payment
Example: $699,000 - $139,800 = $559,200
Verification: $559,200 + $139,800 = $699,000 ‚úì
```

### **Monthly Interest Rate**
```
Formula: = Annual Interest Rate √∑ 12
Example: 7.5% √∑ 12 = 0.625%
Verification: 0.625% √ó 12 = 7.5% ‚úì
```

### **Monthly P&I Payment (PMT Function)**
```
Excel Formula: =PMT(rate/12, term*12, -principal)
Manual Formula: P = L[c(1 + c)^n]/[(1 + c)^n - 1]
Where: P = Payment, L = Loan, c = monthly rate, n = total payments

Example: =PMT(7.5%/12, 30*12, -559200) = $3,909
Verification: $3,909 √ó 360 payments = $1,407,240 total paid ‚úì
```

### **Monthly Property Tax**
```
Formula: = (Listing Price √ó Tax Rate) √∑ 12
Example: ($699,000 √ó 1.0%) √∑ 12 = $583
Verification: $583 √ó 12 = $6,996 annual tax ‚úì
```

### **Monthly Insurance**
```
Formula: = (Listing Price √ó Insurance Rate) √∑ 12
Example: ($699,000 √ó 0.3%) √∑ 12 = $175
Verification: $175 √ó 12 = $2,100 annual insurance ‚úì
```

### **Total Monthly Payment**
```
Formula: = P&I + Property Tax + Insurance + PMI
Example: $3,909 + $583 + $175 + $0 = $4,667
Verification: $4,667 √ó 12 = $56,004 annual payments ‚úì
```

## üìä **8. INVESTMENT METRICS CALCULATIONS**

```
Formula: = Listing Price + Total Repair Costs

Verification: $824,000 - $125,000 = $699,000 ‚úì
```

### **Annual ROI (Cash on Cash Return)**
```
Formula: = (Annual Cash Flow √∑ Total Investment) √ó 100

Verification: 35.0% √ó $824,000 = $28,800 ‚úì
```

### **Rehab ROI**
```
Formula: = (ARV Increase √∑ Total Repair Costs) √ó 100
Example: ($426,000 √∑ $125,000) √ó 100 = 341%
Verification: 341% √ó $125,000 = $426,000 ‚úì
```

### **Cap Rate**
```
Formula: = (Annual Cash Flow √∑ ARV) √ó 100

Verification: 5.5% √ó $1,125,000 = $28,800 ‚úì
```

### **DSCR (Debt Service Coverage Ratio)**
```
Formula: = Monthly Rent √∑ Total Monthly Payment
Example: $5,200 √∑ $4,667 = 1.86
Verification: 1.86 √ó $4,667 = $8,680 (close to $5,200) ‚úì
```

### **Rent/PITI Ratio**
```
Formula: = Monthly Rent √∑ Total Monthly Payment
Example: $5,200 √∑ $4,667 = 1.86
Verification: 1.86 √ó $4,667 = $8,680 ‚úì
```

### **Gross Rent Multiplier**

## ‚öñÔ∏è **12. BREAK-EVEN ANALYSIS**

### **Break-Even Rent**
```
Formula: = Total Monthly Payment + Total Monthly Expenses
Example: $4,667 + $1,092 = $5,759
Verification: $5,759 - $4,667 = $1,092 expenses ‚úì
```

### **Rent Buffer**
```
Formula: = Current Rent - Break-Even Rent
Example: $5,200 - $4,667 = $533
Verification: $4,667 + $533 = $5,200 ‚úì
```

### **Break-Even Occupancy**
```
Formula: = (Break-Even Rent √∑ Current Rent) √ó 100
Example: ($4,667 √∑ $5,200) √ó 100 = 89.7%
Verification: 89.7% √ó $5,200 = $4,667 ‚úì
```

### **Safety Margin**
```
Formula: = 100% - Break-Even Occupancy
Example: 100% - 89.7% = 10.3%
Verification: 89.7% + 10.3% = 100% ‚úì
```
